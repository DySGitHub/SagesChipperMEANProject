Sages Chipper is an online food order service. It was developed for Dynamic Web Development of Year 4 of Software Systems Development. It features various business logic such as add to basket, totalling the basket price, ordering the items and having dynamic specials based on what is in the basket.

Site Features

Menu: This is the menu page where all the items in the menu are shown.

Specials: The specials are loaded dynamically based on what is in the users basket and they are located on the Menu Page.

Basket: This is where all the items added to basket are shown when user wishes to order the need to have item in basket and the information fields entered.

Orders: This is where all the orders and loaded into the system, the Order Date, Total Price and and Information of the User that has Ordered is Shown.
